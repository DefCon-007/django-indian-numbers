django-indian-numbers

GITHUB : https://github.com/narenchoudhary/django-indian-numbers

ISSUES : https://github.com/narenchoudhary/django-indian-numbers/issues

DESCRIPTION

django-indian-numbers is a small reusable Django application for which provides filters for Django templates for formatting numbers in Indian Number System.

FILTERS

  • intcomma_indian
  • floatcomma_indian

EXAMPLES

intcomma_indian

  • {% verbatim %} {{ 0 | intcomma_indian }} {% endverbatim %} will be rendered as {{ 0 | intcomma_indian }}.

  • {% verbatim %} {{ 0.00 | intcomma_indian }} {% endverbatim %} will be rendered as {{ 0.00 | intcomma_indian }}.

  • {% verbatim %} {{ -1 | intcomma_indian }} {% endverbatim %} will be rendered as {{ -1 | intcomma_indian }}.

  • {% verbatim %} {{ 26500 | intcomma_indian }} {% endverbatim %} will be rendered as {{ 26500 | intcomma_indian }}.

  • {% verbatim %} {{ 126500 | intcomma_indian }} {% endverbatim %} will be rendered as {{ 126500 | intcomma_indian }}.

  • {% verbatim %} {{ 1259647552 | intcomma_indian }} {% endverbatim %} will be rendered as {{ 1259647552 | intcomma_indian }}.

  • {% verbatim %} {{ 1259647552.25 | intcomma_indian }} {% endverbatim %} will be rendered as {{ 1259647552.25 | intcomma_indian }}.

  • {% verbatim %} {{ -126500 | intcomma_indian }} {% endverbatim %} will be rendered as {{ -126500 | intcomma_indian }}.

  • {% verbatim %} {{ 126500.25 | intcomma_indian }} {% endverbatim %} will be rendered as {{ 126500.25 | intcomma_indian }}.
